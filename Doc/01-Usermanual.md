## Infrastructure

The runtime data to the respective applications is stored in either `C:\...\...\AppData\Roaming\TerminalChatCLI`, or  `C:\...\...\AppData\Roaming\TerminalChatServer`. 

### Technology

* Programming language: C# with .net
  * I chose C# because I'm most familiar with it.
* Interface: Terminal
  * I'm using the terminal so I can keep the performance as high as possible and so i don't have to justify an ugly design.
* OOP - Object oriented programming
  * To keep a better overview over the project I chose OOP instead of toplevel statements. 

### Platform

* Windows
  * Windows was my OS of choice so far, but I planned to make it multiplatform compatible.
* Compatibility: Linux and MacOS not tested.
  * I haven't tested cross-platform compatibility yet, but support was planned.
