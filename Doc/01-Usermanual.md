## Infrastruktur
 The runtimedata to the respective applications is stored in either `C:\\...\\...\AppData\Roaming\TerminalChatCLI`, or  `C:\\...\\...\AppData\Roaming\TerminalChatServer`. 

### Technology

* Programminglanguage: C# with .net
  * I chose C# because i'm most familliar with it.
* Output: Terminal
  * I'm using the terminal so i can keep the performance as high as possible and so i dont have to justify an ugly design.
* OOP - Object oriented programming
  * To keep a better overview over the project i chose OOP instead of Toplevel statements. 

### Plattform

* Windows
  * Windows was my OS of choice so far, but i planned to make it multiplatform compatible.
* Compatibility: Linux und MacOS not tested.
  * I haven't tested multiplatform compatability yet, though it is planned
